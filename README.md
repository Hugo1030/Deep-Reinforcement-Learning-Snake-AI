# Deep-Reinforcement-Learning-Snake-AI

## 环境依赖

* Python 3.6.2
* TensorFlow 1.3.0
* Pygame
* OpenCV

## 如何运行

* `python play.py` 进行游戏
* `python train.py` 训练模型
